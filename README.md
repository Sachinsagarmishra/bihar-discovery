# Bihar Discovery — Many Faiths. One Timeless Bihar.

Bihar Discovery is a beautifully crafted web portal built to explore the rich spiritual journeys, ancient heritage, living traditions, and cultural roots of Bihar. In collaboration with the Government of Bihar, it guides travelers and pilgrims through diverse faith traditions (Buddhist, Hindu, Jain, and Sikh) with verified resources, travel support, and curated experiences.

## Tech Stack

*   **Bundler & Dev Server:** Vite
*   **Styling:** Tailwind CSS v4 (using `@tailwindcss/vite`)
*   **Language:** Vanilla HTML / JavaScript

## Getting Started

### Prerequisites

Make sure you have Node.js installed.

### Installation

1.  Clone the repository:
    ```bash
    git clone https://github.com/Sachinsagarmishra/bihar-discovery.git
    cd bihar-discovery
    ```

2.  Install dependencies:
    ```bash
    npm install
    ```

### Running Locally

To start the development server:
```bash
npm run dev
```

The app will be available at the local URL printed by Vite (typically `http://localhost:5173/` or `http://localhost:5174/`).

### Building for Production

To create a production build in the `dist` directory:
```bash
npm run build
```
